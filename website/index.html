Why Viz

Your codebase has a shape.
Now you can see it.

Built for engineers who need to understand large, evolving codebases fast.

🌐

Multi-Repo in One View

See all your services and repos as distinct clusters on a single Fibonacci-sphere layout. Spot cross-service dependencies at a glance.

⚡

Real-Time with Claude Code

Run /graperoot-viz inside Claude Code and the graph lights up live as the AI edits your files.

🎯

Typed Edge Families

HTTP calls, Kafka topics, shared DB tables, feature flags — each edge type gets its own color and thickness so signal never gets lost.

🖥️

Runs 100% Locally

No cloud. No telemetry. Your source code never leaves the machine. The Python bridge and graph engine are fully offline.

🔍

Click-to-Explore

Click any node to inspect its dependencies. Click any edge to see its type, confidence score, and source evidence. Fly the camera anywhere.

🍇

Built on GrapeRoot

Uses the same dual-graph engine that powers GrapeRoot Pro's AI context. The graph you see is the same graph the AI uses to understand your code.

Installation

Get running in under a minute

Pick your platform for step-by-step instructions.

1

Download the DMG

Download the latest release for your Mac.

↓ Download .dmg (Universal)
2

Open & install

Double-click the DMG, drag GrapeRoot Viz to Applications.

3

Allow the app to run

macOS Gatekeeper will warn about an unsigned app (until notarized). To allow it:

# Option A — right-click the app → Open → Open Anyway # Option B — run this once in Terminal: xattr -dr com.apple.quarantine "/Applications/GrapeRoot Viz.app"
4

Launch

Open GrapeRoot Viz from Applications. It will ask you to pick a workspace folder (the directory containing your cross_edges.jsonl).

1

Download the installer

Download the Windows NSIS installer.

↓ Download .exe Installer
2

Run the installer

Double-click GrapeRoot-Viz-Setup-x.x.x.exe.

Windows SmartScreen may show a warning — click "More info" then "Run anyway". This is normal for unsigned developer tools.

3

Launch from Start Menu or Desktop

The installer creates a Start Menu entry and optionally a desktop shortcut.

4

Or install silently

# Silent install (no UI) GrapeRoot-Viz-Setup-1.0.0.exe /S
1

One-line install (recommended)

Downloads the AppImage and makes it executable.

curl -fsSL graperoot.dev/viz/install.sh | sh
2

Or download the AppImage manually

↓ Download AppImage (x64)
chmod +x GrapeRoot-Viz-*.AppImage ./GrapeRoot-Viz-*.AppImage
3

Or install the .deb package (Ubuntu / Debian)

sudo dpkg -i graperoot-viz_*.deb
